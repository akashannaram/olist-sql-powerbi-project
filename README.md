# 📊 Olist E-Commerce Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes the Brazilian Olist E-Commerce dataset using **PostgreSQL** and **Power BI** to uncover insights related to sales performance, customer behavior, payment trends, and delivery operations.

The project includes:
- SQL data analysis
- KPI generation
- Data modeling
- Interactive Power BI dashboard creation

---

# 🛠️ Tools & Technologies
- PostgreSQL
- SQL
- Power BI
- DAX
- Power Query

---

# 📊 Dashboard KPIs
- Total Revenue: **16M**
- Total Orders: **113K**
- Total Customers: **99K**
- Average Order Value: **142**
- Delivered Orders: **110K**

✔ Monthly Orders vs Revenue Analysis

---

# 🔍 Key Insights
- São Paulo generated the highest revenue among all states.
- Credit cards were the most preferred payment method.
- Most orders were successfully delivered with a low cancellation rate.
- Revenue showed strong performance during mid-year months.
- Product categories like health & beauty and watches generated high sales.
